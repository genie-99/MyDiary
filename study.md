# Agentic Workflow
AI 에이전트가 주도적으로 작업을 계획하고 실행하는 에이전틱 워크플로우(Agentic Workflow)를 이해하고, 개발 및 협업 프로세스에 적용하여 생산성을 극대화하는 방법을 학습하는 과정입니다. 본 과정에서는 AI 코딩 에이전트인 Antigravity의 활용법부터 마크다운을 통한 효과적인 문서화, Git을 활용한 버전 관리 및 협업 관리까지 실무에 바로 적용 가능한 핵심 도구와 흐름을 배웁니다.

## Course Outline
- Antigravity (An AI coding agent developed by Google DeepMind)
- Markdown (A lightweight markup language for creating formatted text using a plain-text editor)
- Git (A distributed version control system for tracking changes in source code and collaborating)
